西方哲学家和他们的主要观点

发表于 2022-06-11 | 分类于 读书笔记 |

最近读完了《哲学家们都干了写什么》。书中介绍了哲学家们的观点、生平和八卦，介绍了西方哲学从古希腊到20世纪的演化，最后落在人生的意义是什么的问题上。以故事+讨论的形式介绍了各种流派和思想，语言也很活泼，读着使人感觉有趣。本文做一个读后的笔记记录，以免把这些人和观点忘得太快，何况外国名字我本身就记不住。以后有时间也要阅读和梳理中国哲学发展的历程。

宗教和哲学

苏格拉底 - 公元前470-公元前399

  • 古希腊人，打过仗还很猛，喜欢提问题，因为问题太多大家又回答不了，招人烦，被陪审团判死刑
  • 哲学对宗教信仰产生了破坏，但在当时学哲学是长面子的事情

柏拉图 - 公元前427-公元前347

  • 苏格拉底的学生，名字意识是强壮。马其顿打下雅典，亚历山大建立罗马帝国

亚里士多德 - 公元前384-公元前322

  • 柏拉图的学生，亚历山大大帝的老师

保罗

  • 保罗和耶稣同时代。早起是犹太教徒，后面转为基督教。他向犹太人以外的人传播基督教
  • 罗马帝国宗教可以互相杂处，犹太人建立犹太教，基督教从犹太教中发展出来。前者信旧约，后者看新约，旧约记录耶稣出生之前，新约记录之后。基督教是一神教，不让信仰别的神
  • 教父哲学：保罗将哲学的思辨思维应用到基督教传播中，使得基督教获得了很大的传播。他撰写了大量神学文章=》保罗书信，被记录在新约中，此时哲学是传教的工具，发挥不出哲学质疑的能力

后来罗马帝国迫害基督教。但是在之后的权利战争中，皇帝的关键一战获得基督启示，又开始信仰基督教，但是哲学家属于异教徒，被迫害，同时也焚毁了大量哲学书籍。哲学家躲在罗马和东边波斯的边境，希腊著作从拉丁文翻译成叙利亚文。

希帕蒂亚 - 370-415

  • 哲学家，数学家，几何原本。在上述时代被迫害致死

阿拉伯帝国取代波斯，对宗教和文化都是宽容的。阿拉伯百年翻译运动，希腊著作从拉丁文到叙利亚文，到阿拉伯文。后面传到西班牙，从阿拉伯文再到西班牙文。后续在看拉丁文内容，就是从西班牙文翻译到拉丁文的。

13世纪，罗马教廷又开始发挥哲学价值去打击神学异端，开始了《经院哲学》。

托马斯.阿奎那 - 1225-1274

  • 经院哲学集大成者。
  • 靠逻辑推理证明上帝存在：世上万物都需要有因，最初的因就是上帝。上帝是全知全能全善的
  • 罗素对上面的反驳：上帝是否要遵循深恶标准呢？如果遵循，上帝要去符合一定规则那他就不全能；如果不遵循，上帝就无所谓善恶就不是全善的

中世纪的欧洲，教会有着绝对的统治地位，政治上，文化上，军事上等。

马丁.路德 - 1483-1546

  • 神学博士。喜欢思考，产生对教皇权威的怀疑。
  • 认为内心信仰更重要：罗马教会主张外在行为很重要，你只有表现的各项符合教会要求，才能被认为是信仰上帝。马丁路德发现保罗书信“因信称义”，即内心信仰比外在表现更重要，形式不重要。
  • 有了印刷术后，为马丁路德提供了工具。同时他将教徒和教皇的矛盾，转化为日耳曼民族和罗马民族的矛盾（他是日耳曼人）。再者，当时欧洲的经济结构在发生变化，手工业和商业的发展使得城市居民增多，不再像农业更多看天吃饭，对宗教的信仰基础也变薄弱了，为他提供的群众基础。路德用德语写作也使得他的思想传播更广泛，对比其他用拉丁文写作。
  • 基督教开始分裂。路德一派划分为新教，罗马一派为天主教，之前东罗马帝国搞了一个东正教。

加尔文 - 1509-1564

  • 将新教传播到了欧洲其他国家。当上了新教的教皇。
  • 26岁写了《基督教原理》。但是新教也打压自由怀疑的思想。

哲学从宗教中走出

人要思考，这是任何其他动植物的最大区别，思考才能知道是什么，怎么做，为什么，才能找到人生的意义。思考则会产生怀疑。

笛卡尔 - 1596-1650

  • 善于怀疑。包括先哲著作，，人，世界
  • 我思故我在：我是否存在？如果产生了这个疑问，说明是存在的，否则不会有疑问。由此笛卡尔也认为心灵和身体是分开的，二元。
  • 参考《几何原本》的思路，发明了《解析几何》，通过公设来演绎推理出哲学结论。
  • 出生后母亲离世，从小体弱多病。多情。45岁时，在荷兰遇到伊丽莎白公主，频繁通信，教哲学和数学。几年后，瑞典女国王把他接到瑞典去，然后病死了。
  • 安德罗尼克将亚里士多德的著作分为两部分，研究有形的事物《物理学》，研究无形的事物“物理学之后”。日本人参考《易经》“形而上者谓之道，形而下者谓之器”，将其翻译为形而上学。

斯宾诺莎 - 1632-1677

  • 信仰犹太教。父辈在西班牙，斯宾诺莎出生在荷兰。性格温柔，待人宽厚，坚强诚实。喜欢怀疑和思考。被举报到犹太教会。在被迫害后，以磨光学镜片为生。45岁离世。
  • 《伦理学》=》《按几何顺序证明的伦理学》：将不依赖其他事物而存在的绝对存在认为是“实体” => 实体是无限的，唯一的，不可分的，善的 => 世间万物，每个人都是实体的一部分，因为只是一部分，所以都是不完美的 => 那么符合这个实体特征的就是上帝 => 世间万物皆为上帝，我也是上帝的一部分。实体永远存在，我也是实体，可以永远存在。即便肉体消失，也存在，并且是为了善而存在
  • 人与万物合一的中国观点：庄子：天地与我共生，而万物与我为一；北宋张载：宇宙万物一体，侍奉父母爱他人等于爱宇宙；大乘佛教：万物本质为空，即万物都是一样的
  • 上面这些观点，让我想到了爱死机第三季中，在地外星球考察遇险最终与星球融为一体那集

科学与数学的相爱相杀

此时的哲学主流以数学为支撑，但是遭到了科学家的反对。
科学家靠归纳法完成从实验到结论，数学重视演绎推理，科学家认为演绎推理不能带来新的知识，因此轻视它。数学家们认为科学派通过归纳法得到的结论不能保证可靠性。
当时主要分为两派

  • 理性主义：数学拍哲学家，演绎法，严谨，不能产生新的知识
  • 经验主义：科学派哲学家，归纳法，产生新的支持，不能保证结论的绝对正确
    上溯可以追到柏拉图和亚里士多德，一个重视心灵理性，一个重视现实实验。所以亚里士多德说，吾爱吾师，吾更爱真理

约翰.洛克 - 1632-1704

  • 公务员，医生。提出了“天赋人权”等主张。经验主义
  • 刚出生的小孩啥都不知道，哲学家们所说的公设也就不成立。

莱布尼茨 - 1646-1716

  • 德国人。12岁自读拉丁文，17岁哲学硕士，20岁博士。改进计算器能计算加减乘除开发（帕斯卡发明计算器，只能算加法）。喜欢给贵族小姐夫人写信介绍哲学观点[doge]。微积分
  • 理性主义。将和洛克的争论编成《人类理智新论》。
  • 单子论。所有东西向下拆分，最后剩下不占据空间的东西，精神-单子。单子每一个都不同，因此有“世上没有两片树叶是相同的”
  • 补充下帕斯卡：12岁读完《几何原本》，14岁出席法国顶级科学研讨会，16岁出版数据专注，19岁发明计算器，和费马创建了“概率论”。

牛顿 - 1643-1727

  • 物理学家，数学家，天文学家，哲学家，肾虚阿加，炼金术士，世界末日预测者。心眼也小。经验主义，还是洛克的好友。《自然哲学的数学原理》
  • 牛顿的影响之所以大，因为他的理论符合了能够广泛传播的两大标准，1. 准确预测未来 2. 简单好用

此时机械论发展起来，认为所有的一切都是物质决定的，即机械唯物主义。
后来发展出了决定论，虚无主义，享乐主义。既然都是物质决定的，那么所有的事情在最初都已经决定了，都安排好了，是有宿命的。那么我们无论做什么也没啥影响了，是无意义的，要及时行乐。

休谟 - 1711-1776

  • 挑战了机械论和决定论。12岁读大学，23岁完成《人性论》。英国人，经验主义者。公务员，哲学家，历史学家（《英国史》）
  • 他认为之前的争论，都是大家在各自有理的基础上进行，谁也不能把谁驳倒，因为讨论的内容超出了人的经验认知范围，对于不知道就应该承认不知道。
  • 什么是可靠的呢：不依赖于经验的知识，比如三角形这个概念；我们自己感受到的经验。
  • 因果律上述两类都不符合，科学家凌乱了。休谟认为理性主义是独断的，经验主义的归纳法也是不可靠的。
  • 一句名言：你怎么知道明天的太阳会照常升起

康德 - 1724-1804

  • 德国人，一生最远只到过家乡100公里以外的地方。自律甚至可能有强迫症，宅男，大器晚成。50多岁完成了《纯粹理性批判》
  • 康德认为，主客观世界的关系应该颠倒过来。世界的真面目，称为物自体，不可被人们认识的。人民只能通过吱声一个特殊的加工机制感知世界，这个加工机制叫做先天认知形式，而加工之后得到的内容，叫做表象。
  • 之前哲学危机，是休谟对于因果律，对于人类理性能力的怀疑。康德将世界分为两个部分：一部分完全不可知，物自体。一部分可以用理性认知。
  • 局限：奥卡姆剃刀原则：理论应该简洁，不影响结论的部分，应该剔除。不能被认知的物自体，其实是可以被剔除的。再者，康德提出物自体不可被认知，物自体是表象产生的原因，因果规律催在于先天认知形式不在物自体中，则物自体不能应用因果论的话，就不能说物自体是表象产生的原因。而且既然说物自体不可被认知，就不能知道物自体是存在的，且是表象产生的原因

黑格尔 - 1770-1831

  • 1789年法国大革命，读到了康德的作品。1804年，康德去世，他开始写《精神现象学》。黑格尔说世界的本质是绝对精神，之前康德主张的物自体，先天认知形式，表象世界都是绝对精神的一部分。人类研究世界的本质是绝对精神自己观察自己。
  • 辩证法：矛盾无处不在。黑格尔认为矛盾共存，冲突，最后正反融合，升华。这个过程不断循环，绝对精神自己认识自己，人类进步到最终状态，哲学使命完成。
  • 黑格尔认为，哲学就是哲学史，学习哲学就是学习整个哲学史。正是哲学观点的不断变化和迭代构成了哲学。黑格尔至此成为形而上学的巅峰，创造了一个世界上最完备最庞大的形而上学世界。
  • 人生的意义就是活着。人生是什么，就是你活过的这个过程。

理性的陨落

叔本华 - 1788-1860

  • 超级富N代。家庭关系并不和睦，朋友也少。自负，关心名声。宣传禁欲，但是他还挺会享受的。刻薄，性别歧视，看不起女性，但是他也算一个矛盾的舔狗。
  • 崇拜康德，鄙视黑格尔。30岁时发表了《作为意志和表象的世界》。在继承康德的世界观基础上，他认为我们可以认识物自体，因为我们自己就是物自体。之前说用人类理性难以认识物自体，那么叔本华认为，我们可以用直觉直视自己，感受内心，即扪心自问。他把这个条件下的物自体命名为-生命意志。而且万物的物自体是统一的，即生命意志。
  • 悲观主义：他认为人类的理性是生命意志创造出来满足自己欲望的=》人类不过是生命意志实现目的的工具，人没有自由意志。=》怎么反抗？世俗的成功没有意义，要对抗欲望。
  • 局限性：现代心理学表明，潜意识之外，人类理性控制力还挺强。当代的药物或者切除神经等手段也可以控制欲望，可以说用物理的手段控制。逻辑上说生命意志是他对世界万物的观察，这些都是表象，而生命意志是不可被理性描述和认知的，不能用表象论证物自体。

克尔凯郭尔 - 1813-1855

  • 开始家里穷放羊为生，后来发财了，成为有钱人。崇拜自己的父亲。
  • 讨厌黑格尔。他觉得，黑格尔把所有都纳入形而上学的体系，每个人都屈从于必然的历史进程，失去了选择自由。
  • “信仰的飞跃”。无论是对于宗教或者其他内容，是有人类自己选择的自由的，是不能用逻辑描述的，信仰需要跨越这段，需要精神的一跃。帕斯卡的故事：驾驶马车失控，快跌进河里的时候，链接马和车的缰绳断了，这是上帝的神谕，这时虔诚皈依，这是信仰的飞跃。

尼采 - 1844-1900

  • 幼年丧父，被母亲和妹妹照顾。喜欢音乐，喜欢爬上，参过军。是一个温和的智者。崇拜音乐家瓦格纳。但是他和叔本华一样，鄙视女性。
  • 21岁读到了叔本华的《作为意志和表象的世界》。认为真理不存在，还说出了名言：上帝死了。宗教，以前的哲学观点，都是假的，世界是虚无的，但是我们可以选择面对虚无的态度：直面虚无，去战斗，去做你认为有意义的实行，改变自己和世界的命运，去创造意义，去当政治家，哲学家，思想家，艺术家等。
  • 尼采认为，我们身体里存在一种“权力意志”，是一种让自己变得更加强大，有创造力，永不体系的欲望。他也将人和道德分为两部分，战斗的强者和道德，弱者的人和奴隶道德。区别与我们认为人人平等，他强烈认为人不平等。（），他反对的是弱者将躺平道德加到强者身上。对于结果造成的弱肉强食，他说人性如此，所以他也推崇强者，崇拜超人。
  • 他的观点也有现实的基础。经济发展，科学进步，那些虚无的宗教没有了存在基础，而且经济的崛起，强调竞争，资本主义则是资本至上，弱肉强食也很正常。
  • 对于是否平等，我觉得不冲突，平等是从出发点来讲的，他说不平等是从过程和结果来讲的。尼采主张面对虚无的态度，这大概就是正能量的哲学解释了吧，对于我们来讲，要做强者去战斗，不能做弱者躺平。但是战斗的目的并不一定是现在当前的金钱，权力，名声等，而是自己认为有意义的，或者自己创造意义的内容。

科学的支撑

叔本华，克尔凯郭尔，尼采，都动摇了形而上学，认为形而上学和自由意志是矛盾的。但是这时候还有科学，科学是坚持纯理性的，使用归纳法和演绎法。而且这是一战过后，物理学、心理学进步，进化论被发现。科学的发展，给哲学带来了两个影响，一打击了欧洲人对宗教的信心，而是机械论和决定论重新抬头
进化论，一打击了宗教权威；二消除了人的神圣性

罗素 - 1872-1970

  • 贵族之后。长寿，名誉高（文学，社会活动等）。情史极其复杂，结婚又离婚。
  • 《西方哲学史》《数学原理》。逻辑实证主义

维特根斯坦 - 1889-1951

  • 高富富富富富帅，犹太人，父亲经商，家族崇尚文艺。擅长多种乐器喜欢飞行和机械，打过仗，雕刻也很好，当做药剂师，还设计过新的医学试验方法，还编过针对小学生的字典，还有服装设计的天赋。但是家族有自杀传统…中学和希特勒一个学校。和凯恩斯是好朋友。继承大额遗产后，放弃了捐赠给兄弟姐妹，跑去乡下小学教书。但是他殴打过孩子。后来会奥地利，在后来去娇俏读博士研究哲学
  • 逻辑实证主义，罗素的学生。《逻辑哲学论》

逻辑实证主义想要用1. 严格的逻辑符号 2. 经过实际的证明来构建哲学大厦，但是他们失败了。因为到最后，只有两种结论还在范畴之内，一种是桌子是桌子这种，另一种是花式红色的经验描述。维特根斯坦发现，逻辑实证主义到最后得不到一个普遍的理论，说明了理性无法解释世界，指导生活。

同期的，出现了实用主义哲学。实用的，才叫真理。实用主义中心在美国。

波普尔 - 1902-1994

  • 出生在奥地利。
  • 提出了检验科学理论的重要标准：可证伪性。所有的科学理论都是一种假说，没办法完全证实一种科学理论。但是可以提出一个证伪理论，当这个证伪理论被证实前，这个科学理论就是正确的，反之不能证伪的理论就是不对的
  • 证伪主义认为，形而上学追求绝对真理，而绝对真理是不可证伪的，那他就是毫无意义的。

但是其也存在不好用的弊端。

最好用的是建立在经验主义基础上，且实用的。能被观察到，有被大家承认的证据；足够实用，简单。

寻找人生答案

实用主义对于科学很重要，那么对于哲学呢？通过哲学寻找人生的意义，可以打破虚空，让你觉得人生更美好，充实，快乐。

从实用主义来看，我们对于人生的感知，在于各种经历给我们带来的情绪和体验。但是这并没有给出答案。在面临问题，期望得到答案，却发现无解的时候，得到人生是荒诞的？

比如一战后，存在主义兴起，萨特和加缪认为人生就是荒诞的。人生并不像故事，存在开头过程矛盾高潮和结束。最直接的比如死亡，可能随时会来。这是怎么办呢，一种是随波逐流，找一个所有人都相信的意义去面对。存在主义则认为要寻求本质。萨特说“存在先于本质”，本质就是形而上学对于人生意义给出的答案。

思考人生的时候，先不要有任何概念，先看到“我存在”。如何证明我存在呢？我有自由意志。如何体现自由意志呢？我可以自由选择。每一次选择觉定了我的本质是啥，积累起来就是我的人生是什么。那么什么是自由选择呢？是经历过认真思考后的后果，是自己能够接受且真正想做的事情。是高效能人士7个习惯中积极主动的内核，是明代王阳明先生的知行合一。（你看，连起来了，贤哲们的认识连起来了，我读书的内容也连起来了）

书中举了两个例子。

  • 维克多.佛兰克，犹太心理学家，进了集中营，相信自己在里面也有选择
  • 西西弗神话：将巨石推向山顶，落下，再推，如此循环。西西弗认识到了荒谬，但是他还是选择推石头，是尊严和勇气的体现。
    上述也是我们面对人生的需求，做出符合自己意志的选择，追求人生意义，获取人生尊严的映照。

被人类相信的知识是什么，是客观的可以被证实的，是简单实用的。但是对于世界究竟是什么，人生意义究竟是什么而言，没有最终答案。那么，我们在每一个阶段要去做的事情，觉得有意义的内容，都是人生的意义，直到死亡。
所以，人身意义可能是寻找他的过程，不会有最终答案却一直都在。当你不在发问不再求解，看着是不在追寻了，也可能是已经有答案了。
